📝 **Sobre o Projeto**

Construção do meu portfolio para mostrar os projetos em que estou trabalhando no momento e o que estou aprendendo. 


### Tecnologias Utilizadas
1. HTML5
2. CSS3
3. JavaScript
4. Git e GitHub


### O que eu pratiquei?

Neste projeto, foquei em aplicar os seguintes conceitos:

* Troca dinâmica dos temas do site por meio de um botão que muda as cores ao ser clicado;
* Criação de animações usando CSS;
* Estruturação semântica do site;
* Criação de variáveis em CSS;
* Menu de navegação com ícones;
* Utilização de Media Queries para deixar o site responsivo.
